  <h2 className="text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with ft_printf in just a few steps
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the ft_printf repository from GitHub to your local machine.

      ```bash
      git clone https://github.com/KurroLopez/libftprintf.git
      cd libftprintf
      ```
    </Step>

    <Step title="Build the library">
      Compile the library using the provided Makefile. This will generate `libftprintf.a`.

      ```bash
      make
      ```

      <Note>
        The compilation follows 42 School Norminette standards with `-Wall -Werror -Wextra` flags.
      </Note>
    </Step>

    <Step title="Include in your project">
      Include the header file in your C source and link the library when compiling.

      ```c
      #include "ft_printf.h"

      int main(void)
      {
          ft_printf("Hello, %s!\n", "world");
          ft_printf("Number: %d\n", 42);
          return (0);
      }
      ```

      Compile with:
      ```bash
      gcc your_file.c -L. -lftprintf -o your_program
      ```
    </Step>

    <Step title="Use format specifiers">
      ft_printf supports multiple format specifiers for different data types.

      ```c
      ft_printf("Character: %c\n", 'A');
      ft_printf("String: %s\n", "Hello");
      ft_printf("Pointer: %p\n", &variable);
      ft_printf("Decimal: %d\n", 42);
      ft_printf("Unsigned: %u\n", 123);
      ft_printf("Hex (lower): %x\n", 255);
      ft_printf("Hex (upper): %X\n", 255);
      ft_printf("Percent: %%\n");
      ```
    </Step>
  </Steps>
</div>
